# Linear Equation Solver

Solves linear system of equations using numerical methods

## Numerical Methods Used:

1. Gauss-Seidel
2. Gaussian Elimination
3. Gauss-Jordan
4. LU Decomposition
5. Gaussian Elimination With Pivoting
